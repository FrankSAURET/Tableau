
# Tableau
-----
Version 2026.1
Pour Inkscape V 1.x
-----

Dessine un tableau simple avec ou sans coins arrondis.

Optimisé pour la découpe laser.

Utilisation
-----

Copiez les fichiers .inx, .svg et .py dans le dossier des extensions d'Inkscape (généralement sous Windows : C:\Users\nom_utilisateur\AppData\Roaming\inkscape\extensions).
Le dossier exact peut être trouvé dans Préférences/Système.

Redémarrez Inkscape.

> ⚡ **Astuce pratique :**
>
> Cette extension peut être installée et mise à jour automatiquement grâce à l'extension **Màj** :
> **Extensions > Mise à jour des extensions de Frank SAURET...**
>
> Téléchargez Màj ici : [https://github.com/FrankSAURET/Maj](https://github.com/FrankSAURET/Maj)

Le tableau sera disponible dans Extensions > Découpe Laser > Tableau...
